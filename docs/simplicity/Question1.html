Question 1 - Algorithms

You are given an $m\times n$ grid of numbers. Each number in the grid represents the weight of that node in the grid. You have to traverse from the top of the grid to the bottom of the grid while taking the path which offers the least resistance.

  • You are allowed to start anywhere at the top of the grid and finish anywhere in the bottom.
  • You can travel only in three directions the immediate downward-left direction, the immediate downward-right direction and the immediate downward directions.
  • You must find the path which contains the least weight along the journey. Find the total weight of the path with the least resistance.
  • The input is given as a csv file. Here $m = 1000$ and $n = 1000$. All weight values are positive and are lesser than $200$.

    • For example
    Let the given grid be:
    $\begin{array}{cc} 3& 2& 4& 1& 8\\ 41& 5& 11& 6& 15\\ 6& 13& 10& 33& 19\\ \end{array}$

    Consider the element $3$, in the downward direction it can traverse to $41$ and the downward right direction to $5$. Hence it can access only these two elements. Similarly for the element $2$, the only accessable elements are $41$, $5$ and $11$. And for element $11$ the only accessable elemets are $13$, $10$, $33$. Hence the shortest path in the above case is $2 + 5 + 6 = 13$. A $100$ by $100$ element matrix is provided. Please download the csv file as your input.
